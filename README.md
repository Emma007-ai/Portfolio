# Portfolio Website

## Project Overview
This is my personal portfolio website created using **HTML5** and **CSS3**.  
It contains 4 main pages: **Homepage, Aboutpage, Projects page, and Contact page**.  
The website is responsive and adjusts to mobile, tablet, and desktop screen sizes using fluid design and media queries.
